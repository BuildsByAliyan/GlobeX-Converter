# 🌍 GlobeX Converter

A professional and responsive currency converter web application that allows users to convert between multiple currencies with real-time exchange rates. Built using **HTML, CSS, and JavaScript**, this project focuses on simplicity, accuracy, and elegant design.

---

## 🚀 Features

- 💱 Real-time currency conversion
- 🌐 Supports all major currencies
- 🧭 Fully responsive layout
- 🎨 Clean and modern UI
- ⚡ Fast and lightweight
- 🔒 Secure and user-friendly interface
