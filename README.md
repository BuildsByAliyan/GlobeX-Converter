## ✨ Overview
<div align="center">
**GlobeX Converter** is a sophisticated, real-time currency conversion platform. Built with a **Glassmorphism UI**, it delivers instant exchange rates through an intuitive interface that works flawlessly on **Mobile (393x851)** and Desktop.
</div>

---

## 🚀 Key Features
<div align="center">

| Feature | Description |
|:-------:|:------------|
| ⚡ **Real-Time** | Instant calculations with live API exchange rates |
| 📱 **Responsive** | Perfect fit for Mobile (393x851) and Desktop |
| 🎨 **Classy UI** | Modern frosted-glass aesthetic with smooth animations |
| 🔄 **Dynamic** | Switch between 150+ global currencies instantly |

</div>
