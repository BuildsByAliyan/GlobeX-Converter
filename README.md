## ✨ Overview
**GlobeX Converter** ek sophisticated, real-time currency conversion tool hai. Isay khaas taur par **Mobile (393x851)** aur Desktop dono ke liye optimize kiya gaya hai. Iska interface **Glassmorphism** design par mabni hai, jo isay aik classy aur fintech look deta hai.

---

## 🚀 Key Features
* 💱 **Real-Time Rates:** Live API integration ke zariye har lamha up-to-date exchange rates.
* 📱 **Perfectly Responsive:** 393x851 dimension par bina kisi scrolling ke fit baithta hai.
* 🎨 **Premium UI:** Frosted glass effect aur smooth animations ke sath modern design.
* 🌐 **Global Support:** Dunya ki tamam bari currencies (150+) ka support.
* ⚡ **Lightning Fast:** Bohat hi halka aur tezi se load hone wala code.

---

## 🛠️ Technology Stack
- **HTML5:** Semantic structure ke liye.
- **CSS3:** Glassmorphism aur responsive grid layout ke liye.
- **JavaScript (ES6):** Dynamic calculations aur API fetching ke liye.
- **ExchangeRate API:** Real-time data sync karne ke liye.
